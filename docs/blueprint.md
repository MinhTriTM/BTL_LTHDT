# **App Name**: Student Manager Pro

## Core Features:

- Student Creation: Allows creation of new student records with information such as ID, full name, date of birth, and GPA.
- Student Listing: Displays a comprehensive list of all students with their key information.
- GPA Calculation Assistant: A tool that uses AI to suggest appropriate GPA ranges based on entered grades. Can incorporate reasoning for borderline cases.
- Student Search: Enables searching for students by ID or name.
- High Achiever Highlighting: Highlights students with a GPA greater than 8.0.
- Top Student Finder: Identifies the student with the highest GPA.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for a sense of trust and education.
- Background color: Light blue-gray (#ECEFF1) for a clean and professional look.
- Accent color: Muted green (#A5D6A7) to highlight key actions or information.
- Headline font: 'Space Grotesk' sans-serif, for a modern and slightly techy look.
- Body font: 'Inter' sans-serif, for a clean, readable appearance in longer text.
- Use simple, line-based icons for navigation and actions.
- Use subtle animations for feedback on user actions.
